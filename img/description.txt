Конфигуратор состоит из трех частей
1. HTML страница с полями

data-alt - описание параметра;
data-price - цена;
data-value - ззначение параметра (по нему составляется обьект машины);
data-prop -  значение параметра (по нему просграма понимает в какое поле обьекта нужно поместить значение параметра);
data-index - индек по которому определяется какие поля нужно перебрать, снять с них выделение и запомнив индекс выделеного поля подсветить его;

2. conf-class - тут описаны классы и методы. Они должны быть универсальными и подходить под все случаи.
3. conf-car - файл где из класов и событий создается обьект автомобиля. 
4. tegs - тут будут определены все теги и сам обьект с которым ведется робота.
